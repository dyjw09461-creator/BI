# 积分兑换平台 BI 综合仪表盘设计

## 目标

实现商务智能实训要求的 BI 核心功能：商家会员统计分析、礼品统计分析、订单统计分析、数据仓库模型设计、ETL 抽取和自定义报表展示。最终页面是一个精美的 Flask + ECharts 单页仪表盘，不翻页，不展示销售模板字段。

## 业务范围

数据源只使用 `BIDemo_AccumulateCoin`，数据仓库使用 `BI_GoldCoin_DW`。01 业务系统和 03 BI 大屏合并理解为一个积分平台 BI 系统：业务页面用于查看和维护业务数据，首页用于展示 BI 仪表盘。

## 分析主题

基础主题：商家统计分析、会员统计分析、礼品统计分析、订单统计分析、商家会员统计分析。

扩展主题：商品积分统计分析、积分码使用统计分析、积分流水统计分析、地域统计分析、时间/小时活跃分析、礼品库存消耗分析。

## 总线矩阵扩展

维度：时间、小时、商家、商品、会员、礼品、积分码、积分方式、订单状态、地域。

度量：商家数、会员数、商品数、礼品数、订单数、订单积分、积分收入、积分支出、积分码数量、积分码使用率、商家会员数、礼品兑换次数、礼品库存消耗、地域会员数、地域订单数。

## 仪表盘

顶部展示数据源、DW、ETL 更新时间和业务导航。KPI 展示会员、商家、礼品、订单等核心指标。图表区使用订单趋势折线图、积分码环图、商品贡献横向条形图、商家会员横向条形图、礼品兑换明细表、地域排行条形图和业务观察卡片。

## UI 方向

使用 `frontend-design` 方向：浅色科技经营驾驶舱，白色玻璃面板、浅蓝网格背景、精细边框、克制动效。避免黑金模板、销售模板、营销页风格和无关文案。
